# OpenGL ゲームライブラリ


## シェーダー描画について
* コンストラクタに引数なしでインスタンスを生成すると既定のシェーダーファイルが選択されて自動で有効無効になり表示可能なお既定じゃないシェーダーファイルを選択するときは手動で有効無効を設定する必要あり
